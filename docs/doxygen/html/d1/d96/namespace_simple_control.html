In Arduino, Serialized class is alias of Serialized_Arduino, the other side Serialized is alias of Serialized_STL

Serialized_Arduino provides functions for Serialized :: value_type array, in addition Serialized_STL provides functions for some types, ex. std :: vector<Serialized :: value_type >
